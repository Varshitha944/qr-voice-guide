# Smart QR-Based Multilingual Voice Guidance System

## Project Overview

The Smart QR-Based Multilingual Voice Guidance System is a web-based application developed using Flask that provides voice instructions in multiple languages through a QR code-enabled interface. Users can scan a QR code, access the web application, and listen to instructions in their preferred language.

## Features

* QR Code-based access
* Multilingual voice guidance
* Text-to-Speech functionality
* User-friendly web interface
* Admin panel for managing instructions
* Cloud deployment using Render
* Version control using GitHub

## Technologies Used

## Project Workflow

1. Admin adds instructions in multiple languages.
2. Instructions are stored in the SQLite database.
3. Users scan the QR code.
4. The web application opens.
5. Users select their preferred language.
6. The system reads the instruction aloud using Text-to-Speech.
